使用 Node.js 和 Mineflayer 构建全自动 Minecraft 挖矿机器人

2026-02-25
  • JavaScript
  • Node.js
  • 自动化

在 Minecraft 服务器中，挖矿往往是一项重复且耗时的工作。本文将介绍如何使用 Node.js 和 Mineflayer 库开发一个自动挖矿机器人（Bot）。这个机器人不仅能够自动寻找并挖掘珍贵矿物，还支持聊天指令控制、自动寻路回家以及物品存储功能。

项目概览

本项目是一个基于 mineflayer 的 Minecraft 机器人，主要功能包括：

  • 自动挖矿：自动寻找并挖掘钻石、绿宝石等珍贵矿石。
  • 指令控制：通过游戏内聊天窗口发送指令控制机器人。
  • 自动寻路：使用 mineflayer-pathfinder 插件实现复杂地形的自动寻路。
  • 物品管理：支持查看背包、丢弃物品以及自动存入箱子。
  • 断线重连与死亡恢复：具备基本的错误处理和自动恢复机制。

环境准备

在开始之前，请确保你已经安装了以下环境：

  • Node.js (建议版本 v14+)
  • 一个 Minecraft 服务器 (支持 Java 版)

核心代码解析

1. 项目入口 (bot.js)

bot.js 是机器人的入口文件，主要负责初始化机器人实例、加载插件以及注册事件监听器。

2. 配置文件 (conf.js)

为了方便管理，我们将机器人的配置项提取到了 conf.js 中。这里定义了管理员名单、目标矿物以及其他行为参数。

3. 指令处理 (lib/commands.js)

机器人通过监听 chat 事件来接收指令。为了安全起见，我们只响应配置在 ops 列表中的管理员发出的指令。

支持的主要指令包括：

  • run: 开始自动挖矿
  • stop: 停止当前行动
  • bag: 列出背包中的物品
  • home: 传送到管理员身边 (需要服务器开启 /tp 权限)
  • save: 自动寻找附近的箱子并存放物品

4. 自动挖矿逻辑 (lib/mining.js)

这是机器人的核心逻辑。mineLoop 函数是一个持续运行的循环，它不断寻找附近的珍贵矿物并前往挖掘。

逻辑解析：

  • **findNearestValuableBlocks**：利用 mineflayer 内置的 findBlock API。它在指定的 radius 半径内（通常为 50 格）进行 3D 空间搜索。
  • **匹配器 (Matching)**：通过回调函数过滤方块，只有在 conf.js 中定义的矿石才会被选中。
  • 循环机制：mineLoop 采用 while 循环，确保机器人在挖掘完一个矿石后，立即开始下一次搜索，实现全自动作业。

深入算法与技术挑战

除了基础的连接和指令，本项目还涉及了一些有趣的“算法”和物理模拟技巧，这使得机器人更像一个真实的玩家。

1. 物理模拟：跳跃搭桥 (Fast Towering)

在 mining.js 中，我们实现了一个 fastTower 函数，让机器人能够像真人玩家一样“原地起跳并在脚下放路”。这需要精确的物理状态同步。

解析：

  • 物理监听：通过监听 physicsTick 事件（每秒 20 次），实时监测机器人的速度向量。
  • 顶点检测：当 velocity.y 几乎为零时，意味着机器人到达了跳跃的最高点。此时放置方块的成功率最高，且不容易被方块“卡住”。

2. 动态避障：Head Crack 逻辑

在进行移动或起跳前，机器人会预先检查头顶空间，这就是 headCrack 算法：

这种“先清理，后行动”的防御式编程，极大地提高了机器人在复杂地形（如洞穴、狭窄矿道）中的生存率。

3. A* 寻路与代价评估

机器人依赖 mineflayer-pathfinder 实现 A* 寻路。在 events.js 中，我们对其进行了深度配置：

  • **挖掘代价 (canDig)**：允许路径规划算法考虑通过挖掘方块来开辟路径，而不仅仅是走空地。
  • **原地搭桥 (allow1by1towers)**：允许机器人通过脚下垫方块（搭桥）来垂直上升。
  • **流体保护 (dontCreateFlow)**：防止机器人在挖掘时意外触发岩浆或水流。

4. 矿物搜索算法

在寻找矿物时，我们使用了简单的空间索引搜索。通过 bot.findBlock 配合 matching 函数，可以在 O(N) 时间复杂度（N 为周围方块数）内快速定位最近的目标。

功能演示

  1. 启动机器人：运行 node bot.js，机器人将加入服务器。
  2. 开始工作：在游戏中对机器人输入 run，它将开始自动寻找并挖掘配置中的矿石。
  3. 查看状态：输入 bag 可以查看机器人当前的背包物品。
  4. 停止工作：输入 stop 让机器人停下脚步。

总结

通过 Mineflayer，我们可以用几十行代码实现一个功能强大的 Minecraft 机器人。本项目的代码结构清晰，通过模块化设计（events, commands, mining 等）使得功能易于扩展。你可以在此基础上增加更多功能，例如自动打怪、自动种植或者建筑功能。

注：请遵守服务器规则，不要在禁止使用 Bot 的服务器上运行此程序。

使用 Python 自动化下载 Gut and Liver 期刊 PDF

核心功能模块

1. 配置请求头和 CSV 文件锁

为了防止被反爬机制阻拦，我们自定义了 headers，并且使用线程锁 csv_lock 确保多线程写 CSV 时不会冲突。

3. 构造文章列表页请求参数

4. 下载 PDF 并计算 MD5

核心点：

  • 文件按 MD5 命名，避免重复。
  • 使用 stream=True 分块写入，减少内存占用。
  • 下载完成后将信息写入 CSV。

5. 处理每篇文章信息

解析文章标题、作者、日期、摘要、PDF 链接等信息，并调用 process_info 下载 PDF。

6. 多线程抓取所有文章

使用 ThreadPoolExecutor 加速多文章抓取与下载，避免阻塞。

总结与注意事项

  1. 使用线程锁保护 CSV 写入，保证多线程安全。
  2. PDF 文件以 MD5 命名，防止重复下载。
  3. 可根据需要调整 max_workers 控制并发量。
  4. 解析 HTML 时，需要处理可能不存在的标签（如摘要或 PDF 链接）。
  5. 建议加上异常处理和重试机制，保证稳定运行。
